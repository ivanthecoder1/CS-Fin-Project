How to get started

  • Find places that will teach you how to code: summer programs, coding club, a cs class offered at your school, or the internet.
  • Choose your starting language: We recommend Python due to its beginner friendly syntax.
  • Find resources/tutorial: Start learning the fundamentals from a variety of credible sources. However, don’t stay in tutorial land forever.
  • Tackle projects: Build simple projects. It doesn’t have to be complex, it could be as simple as a calculator or a madlib. Get your hands dirty.
  • Learn how to google: Build a habit of relying on google to aid you with bugs or syntax that you don’t know.

Basic Coding Vocabulary

  • Programming language: A formal language comprising a set of strings that produce various kinds of machine code output. Each language is uniquely designed for various purposes. Ex: Python is used for data science, Javascript is used for web development, C++ is widely used for game development.
  • Syntax: Refers to the rules that define the structure of a computer language. The structure of the symbols, punctuation, and words may be different. Each computer language has its own unique syntax.
  • Command line: Known as a text interface for your computer. It's a program that takes in commands that you type in, which is then passed to your computer's operating system to run. You can navigate through files and folders on your computer with your command line.
  • Variable: A value that can change, depending on conditions or on information passed to the program. A variable can hold the results of a calculation, database call, results of a database query, or other value.
  • Boolean: a data type with two possible values: true or false
  • String: a sequence of characters. Ex: “Hello world”
  • Floats: a variable type that is used to store floating-point number values or decimals. Ex: 4.00001
  • Conditionals: A way for computers to make decisions based on conditions. If/Else are A common form of conditional statements in programming that tells the computer that if the condition is true, do this. Else, if the condition is false, do another thing.
  • Loops: A programming element that repeats a portion of code a set number of times until the desired process is complete. The two most popular loops are the for and while loop.

About Us

WeRise is a educational website designed to connect young POC to important skills such as computer programming and finance, as well as checking in on their mental health. Immediate action is needed in order to diversify the workplace in the future. We created this site in hopes of introducing young people of color, such as ourselves, to new opportunities while nurturing our overall well-being.
